# SLURP utilities used in this repo

Yes — this SLURP workflow was adapted from SpeechBrain's SLURP recipe:
https://github.com/speechbrain/speechbrain/tree/main/recipes/SLURP

## What is actively used now

This repo currently keeps the **active** evaluator used by `run_slurp.sh`:

- `evaluate_qwen.py`: evaluates `predictions.jsonl` generated by
  `finetuning/qwen3_asr_test.py` against qwen-format ground truth
  `data-json/slurp/{dev,test}.jsonl`.

## Why legacy files were removed

Earlier local copies (`prepare.py`, `evaluate.py`, `audioio.py`, and `metrics/`) were
historically derived from external SLURP recipes (including SpeechBrain-style structure),
but they were not directly wired to this repository's current Qwen3-ASR I/O format.
To reduce confusion and maintenance cost, the active path is now:

- `local/prepare_slurp_jsonl.py` for preprocessing
- `local/slurp/evaluate_qwen.py` for evaluation
- `run_slurp.sh` as the stage runner
